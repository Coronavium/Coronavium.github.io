Coronavium Cup

What is Coronavium?

Coronavium is a decentralised digital currency with near-instant transaction speeds and negligible transaction fees built upon Proof of Stake 3.0 (PoSV3, BPoS) as introduced by the Corinavium development team.

What is Coronavium Cup?

Coronavium Cup is the name of open source software which enables the use of this currency. It takes Coronavium to the next level by building upon Bitcoin Core 0.13.2 with some patches from newer Coronavium Cup versions to offer performance enhancements, wider compatibility with third party services and a more advanced base.

How to upgrade?

If you are running Blackcoin Legacy wallet v1.2.x:

You need to download the blockchain and send your funds to the new wallet.

If you are running Blackcoin Lore wallet v2.12.x:

Technically you can point Coronavium Cup installation to your data folder, but it is recommended to create a new installation in the default data folder. The same for your funds, it is technically possible to use the same wallet.dat file, but it is better to use the wallet created by new installation. Send your funds to this new wallet.

If you are running Coronavium Cup wallet v2.13.x:

You can upgrade just by replacing binaries, but backup your wallet first (wallet.dat)
